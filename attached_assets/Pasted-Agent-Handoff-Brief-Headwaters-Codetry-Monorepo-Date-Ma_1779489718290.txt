THE CONSTITUTIONAL FRAMEWORK (three layers, load-bearing)
Layer 1 — Naming (vocabulary):
Saltbox, Eave, Workbench, Compost — constellation/tool names, not zone labels.

Layer 2 — Trust-Gradient Spine (governs all schema):

Z0 Dwelling/Household → Z1 Circle → Z2 Home Range →
Z3 Community → Z4 Trade → Z5 Wild
Layer 3 — Practitioner Energy (display only):
Dual labels: "Household / Afloat", "Home Range / Build now", etc.

THE EAVE RULE — poured concrete, non-negotiable:
The Eave is the hard seam between Z1 (Circle — private household identity) and Z3 (Community — XRPL wallet). No table, foreign key, join, query path, or stored reference may ever connect a Z3 wallet address to a Z1 household record. Any feature that would create such a path must be refused or redesigned.

THE KITCHEN TABLE sits in Z2 (Home Range / Workbench). Deliberation space. Curtains drawn by default.
